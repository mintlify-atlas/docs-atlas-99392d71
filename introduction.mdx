# An Anime Game Launcher

A modern, feature-rich launcher for running anime games on Linux. Built with GTK4 and libadwaita, this launcher provides seamless game management with support for multiple Wine runners, DXVK, and various quality-of-life features.

## Command-line options

The launcher supports several command-line arguments for advanced usage:

```bash
# Enable debug logging output
anime-game-launcher --debug

# Run the game directly if ready
anime-game-launcher --run-game

# Force run the game (bypass pre-download checks)
anime-game-launcher --just-run-game

# Disable verbose tracing in stdout
anime-game-launcher --no-verbose-tracing

# Switch to a specific session
anime-game-launcher --session <session-name>
```

<Note>
  The launcher automatically creates its data directory at `~/.local/share/anime-game-launcher` and cache directory at `~/.cache/anime-game-launcher` on first run.
</Note>

## Project status

This project is in maintenance mode and is considered feature complete. It continues to receive updates with bugfixes and support for new versions of the game. The An Anime Team is working on a [universal launcher](https://github.com/an-anime-team/anime-games-launcher) that will unite all anime game launchers in the future.

## System requirements

- Linux operating system (any distribution)
- GTK4 and libadwaita
- Sufficient disk space for game installation
- Internet connection for downloading game files and updates
